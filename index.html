Experience

I had the amazing opportunity to intern at Microsoft working on the Azure Storage team. While here I learned a lot about distributed systems and what goes into every layer when storing data. I specifically worked on optimizing the algorithm for cold data in zone-redundant storage. The way the infrastructure is set up, there is an optimization that can cut bandwidth cost by 2/3 by conducting erasure coding in a multi-step fashion.

Starting in January 2020, I was able to work at Amazon Robotics as a software engineer co-op. Here, my work consisted of adding features to a gradle CLI project that is used across several teams and creating a standalone image generator for a computer vision training set. The simulation was built in Unity and bundled up into a GPU-enabled Docker image that then was created as a queryable AWS Batch process.

For my first co-op I had the opportunity to travel to Tokyo, Japan and work as a software engineer at TableCheck. While there, I worked on several backend and frontend projects focusing on implementing a microservice architecture.
