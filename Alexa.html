Text Based Adventure Alexa Skill

This project was completed by Cole Pobanz, William Tablan, Anthony Corigliano, and myself in our AP Computer Science class. Our final for the class was very open ended and required that we come up with our own project. We decided to make an Alexa text (voice) based adventure game. Because we learned coding in Java the skill would have to be written in Java. The whole process was super confusing at first because deploying a Java skill through Amazon Web Services involved the use of Maven which we were clueless about. But through research and a lot of trial and error we figured it out. While we never got the skill published, it still allowed me to put a project under my belt and was a great experience learning web deployment and gaining further knowledge in the field of coding.
